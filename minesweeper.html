MineSweeper Game

Built the MineSweeper game using Java. Supports randomized placement of mines after every turn while ensuring a winnable game. Has a visually appealing GUI and shows counters for count clicks and flagged mines. The goal is to flag all the mines and uncover all the tiles.

Link to github repo

(Available upon request)

Features

  • Clicking on a random tile will either result in game over if a mine is clicked, or will lead to a cascade where all tiles without neighboring mines will uncover.
  • Left click to flag a mine.
  • Left click on a flagged mine will get rid of the flag.
  • The numbers on the tiles represent their number of neighboring mines, also color coded.
  • The unflagged mines counter shows how many mines you have left to flag.
  • The covered tiles counter shows how many tiles are left covered.
  • The reset button restart a new screen with a new game.
  • The board size is configurable, the number of mines placed depend on the baord size to ensure a playable game.

Technologies Used

Java, Ecllipse IDE
