Co-op Application Website

This project is a game with a goal of lighting up each piece and connecting the entire board. Uses breadth first search to determine which squares should be lit up. Uses Kruskal's algorithm with union-find to randomize the placement and orientation of pipes for every new game, ensuring optimality and a playable game.

Link to github repo

(Available upon request)

Features

  • Each piece can be clicked on to be rotated.
  • If a piece is connected to the powerstation it will light up, the goal is to light up all pieces.
  • The powerstation can be moved using arrow keys, it can only move along connected paths.
  • The board size is configurable, depending on the board size, the number of clicks allowed changes.
  • If you run out of clicks and every piece is not connected the game is over.

Technologies Used
