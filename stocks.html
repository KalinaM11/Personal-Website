Stocks Application

This project is a stock simulator platform that allows you to buy/ sell stocks, make portfolios, save them and reload them and veiw the portfolio's/share's value on a given dates. Creating using Java and the Model-View-Controller (MVC) model. Also used command-design pattern and encapsulation. Used Alpha Advantage API for accurate real-time stock data.

Link to github repo

(Available upon request)

Features

  • Buying and selling shares.
  • Looking up the value of shares on a given date.
  • Creating a portfolio.
  • Saving a portfolio and reopening it after closing the application.
  • See your portfolio's value on a given date.
  • Sell/buy stocks from your portfolio
  • Supports a text-based UI and a GUI (the text-based UI has additional features such as viewing a graph, crossover period, and moving average)

Technologies Used

Java, IntelliJ IDE
